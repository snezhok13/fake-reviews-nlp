# Fake Reviews NLP

Дипломный проект (Нетология): автоматическое выявление фейковых отзывов на маркетплейсах.

## Задача
Сравнить классические ML- и DL-подходы для классификации подлинных и накрученных отзывов.

## Стек
Python, pandas, scikit-learn, LightGBM, TensorFlow/Keras, TF-IDF, GloVe, LIME
